# gi-cli

[![David DM](https://david-dm.org/noxan/gi-cli/status.svg)](https://david-dm.org/noxan/gi-cli)

Git issues command line interface.

## Config file

Make sure to have a configuration file at `$HOME/.gi.yaml`. Here an example for
its content:

    ---
    github.com:
      token: <AUTH-TOKEN>
